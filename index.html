Bluetooth-Controlled LED Circuit

Hardware C++ Microcontroller

Overview: A wireless control system utilizing a mobile device to toggle physical hardware.

Components Used:

  • Arduino Nano
  • HC-05 Bluetooth Module
  • Breadboard, LEDs, and 220-ohm Resistors

How It Works:

This project establishes serial communication between a smartphone and the microcontroller. By sending specific string commands from a mobile Bluetooth terminal, the board parses the incoming data and toggles the corresponding digital pins to control the LED states.
